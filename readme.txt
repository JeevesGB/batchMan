What is this? 

A script to scan a folder & subfolders then output the contents to a .txt file.

I will be adding the ability to batch rename files to what you have in list.txt and a script to convert .mp3 files to .wav (you WILL need ffmpeg.exe for this)


How to?

First set your directory in the scan script and then run scan.py 

You will now see list.txt this will contain all of your folders and files. 

Next if your audio files are in .mp3 format you will want to go to the convert.py script and set the relevent directorys 

Run convert.py and once it's done you will see your files in your chosen output directory.

Now you want to go to rename.py and set your directorys then run the script this will rename all the files within your input directory and save them to your output directory.
